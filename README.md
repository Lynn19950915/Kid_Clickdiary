### 專案簡介

幼兒點日記是一套專案管理系統，讓用戶在毋需撰寫程式碼之下生成問卷版型，

並以專案為單位執行測試填寫、url 管理及上線追蹤等工作。

期望藉此降低網頁生成的門檻，也達到統合管理的高效成果。
